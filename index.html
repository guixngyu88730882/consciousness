VISION
01 愿景

如果每个NPC
都能记住你？

SENTIENCE 弥合了脚本行为与真正智能之间的鸿沟。通过将本地运行的大型语言模型直接集成到 GTA V 运行时中，每个行人、店员、帮派成员和路人都成为了一个有思想、有情感的实体，拥有持久的记忆、情感状态，以及应对惊喜的能力。

记忆

NPC会记住过去的相遇，建立跨越游戏会话的持久关系。

情感

动态情感建模——恐惧、信任、好奇、愤怒——驱动真实的反应。

隐私

100% 本地推理。你的对话永远不会离开你的机器。零云端，零追踪。

FEAT
03 特性

它们能看、能思考。
它们会记住。

01

持久记忆

每个 NPC 都维护着一条记忆链。帮助某人一次——几周后它们仍会认出你。伤害某人——它们绝不会忘记。

02

自然对话

自由输入交谈。NPC 会根据上下文和人格给出恰当的回复——而不是预设的台词。

03

情感 AI

多轴情感模型（信任、恐惧、快乐、愤怒、好奇）塑造每一次回应和行为选择。

04

涌现行为

NPC 会形成观点、传播谣言、做出自主决定。整座城市变成了一个活生生的社交网络。

05

模组友好 API

开放的 C# API 让其他模组作者可以接入 SENTIENCE。在活生生的 NPC 之上构建任务、故事或系统。
